1) Model - управляет данными и уведомляет View об изменениях через
Наблюдателя. В этом модуле традиционно находится вся логика, связанная
с хранением и обработкой данных

2) View - представление данных пользователю. Получает уведомление об
изменении данных, подписываясь на model

3) Controller - получает ввод от пользователь и передает команды в Model
